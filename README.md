# Big Mart Sales Analysis & Prediction

## Overview
This project analyzes sales data from Big Mart to identify key drivers of product sales and build predictive models.

## Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Modeling
- Model Evaluation
- Business Insight

## Tools
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Key Findings
- Item_MRP is the strongest predictor of sales
- Outlet_Type significantly impacts performance
- Tree-based models outperform linear regression
